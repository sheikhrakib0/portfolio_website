    About this Project

    A Retrieval-Augmented Generation (RAG) web API built using FastAPI, LangChain, and Google Gemini. This project allows you to upload documents, store their embeddings, and ask natural language questions about the uploaded content. The system retrieves relevant information from the documents and generates accurate responses using advanced language models. Key features include document upload, embedding storage, and a question-answering interface powered by Google Gemini.
    Features:
    * Upload documents (PDF, DOCX, TXT)
    * Store document embeddings in a vector database using FAISS
    * Ask questions in natural language and get gemini-generated responses
    * Get accurate answers based on document content
    * Built with FastAPI for high performance and scalability
    * Uses LangChain for seamless integration of language models and vector databases
    Tech Stack:
    * Backend: FastAPI
    * Framework: LangChain
    * LLM Model: Google Gemini
    * Vector Database: FAISS
    * Deployment: Render.com
    * Frontend: HTML/CSS/JavaScript
